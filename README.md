# Постулат Логоса — PWA

## Файлы

| Файл | Назначение |
|------|-----------|
| `index.html` | Главная страница с выбором версии |
| `postulate-column.html` | Версия с колонкой (n квадратов N×N в столбец) |
| `postulate-sudoku.html` | Версия судоку (поле n²×n², два фазы решения) |
| `manifest.json` | PWA манифест |
| `sw.js` | Service Worker (кэш, офлайн) |
| `icons/icon-192.png` | Иконка 192×192 |
| `icons/icon-512.png` | Иконка 512×512 |

## Установка на сервер

Скопируйте все файлы в корень сайта с HTTPS.
PWA установится автоматически при открытии в браузере.

## Локальный запуск

```bash
python3 -m http.server 8080
# открыть http://localhost:8080
```

## Офлайн

Service Worker кэширует все файлы при первом открытии.
После этого приложение работает без интернета.
