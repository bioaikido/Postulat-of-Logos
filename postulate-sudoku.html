Постулат Логоса

В поле n²×n² квадратов, заполненных картами 1..n², цепочка решений конфликтов однородна: каждое звено идентично первому по длине шага и направлению перестановки.

Правила

  1. Поле: n²×n² ячеек, разбитых на n² квадратов n×n (как судоку).
  2. Колода: n² наборов карт 1..n². 80% поля заполнено по правилам судоку.
  3. Запрет: нельзя поставить карту, если значение уже есть в том же доме, строке или колонке.
  4. Конфликт по колонке — разрешается горизонтальными перемещениями внутри домов.
  5. Конфликт по строке — разрешается вертикальными перемещениями внутри домов.
  6. Сначала решаются все конфликты по колонкам, затем по строкам.
  7. Цепочка однородна: шаг и направление фиксируются при первом ходе.
  8. Правило 4: последняя активная карта не двигается — двигается другая.
