Постулат Логоса

В ряду из n квадратов n×n, заполненных картами 1..n², цепочка решений конфликтов однородна: каждое звено идентично первому по длине шага и направлению.

Правила

  1. Колода: n наборов карт 1..n².
  2. 75% поля заполнено по правилам судоку (нет дублей в квадрате и колонке).
  3. Запрет: нельзя ставить карту, если такое значение уже есть в квадрате.
  4. Конфликт: дубль в колонке — создаётся игроком намеренно.
  5. Solve: решает конфликты горизонтальными перестановками внутри квадратов.
  6. Back / Fwd: прокрутка ходов решения.
